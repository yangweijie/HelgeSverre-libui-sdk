## hook
Think PHP is only for the web? [[cue:web]]Meet yangweijie slash ui two — native desktop GUIs for PHP 8.5 and above. [[cue:gui]]No web server. No Electron. [[cue:noelectron]]Just real, native applications.

## architecture
Built on libui-ng via PHP's Foreign Function Interface, [[cue:ffi]]your code talks directly to native operating system controls. [[cue:controls]]Cocoa on macOS. GTK on Linux. Win32 on Windows. [[cue:platforms]]No browser engine, no JavaScript runtime.

## features
Rich form fields, modal dialogs, color and font pickers. [[cue:pickers]]Custom-drawn widgets like circle progress bars, toggle switches, and status indicators. [[cue:widgets]]Plus an embedded WebView for HTML inside your desktop app.

## devx
The API feels natural to PHP developers. [[cue:api]]Declarative menus, composite layouts, a powerful patching system. [[cue:composite]]Create a fully functional window in five lines of code. [[cue:five]]One Composer install, and you're building desktop apps on every platform.

## crossplatform
One codebase. Three platforms. [[cue:three]]macOS, Windows, Linux — all from a single Composer install. [[cue:single]]Build once, run everywhere PHP runs.
